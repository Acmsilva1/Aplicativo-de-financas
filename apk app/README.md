# APK App

Pasta reservada para a distribuicao por APK fora da Play Store.

## Objetivo

Guardar a saida final do app Android empacotado para instalacao direta, via link, drive, site proprio ou repositorio de releases.

## Status atual

- A versao web continua hospedada na Vercel
- Esta pasta ainda nao contem um build Android pronto
- Quando o APK for gerado, ele deve ser colocado aqui junto com os arquivos de release

## O que entra aqui

- `app-release.apk`
- `release-notes.md`
- `checksums.txt`
- capturas de tela para distribuicao

## Passos futuros

1. Gerar o APK em ambiente Android.
2. Testar em um celular real.
3. Publicar o arquivo em um canal de distribuicao proprio.
4. Registrar a versao liberada.
